## About project
A set of console mini-games. Project to learn PHP environment setup, maintainable architecture and code naming.

## Setup
```
$ composer global require kulikov-rv/brain-games
```

## Games list
* brain-even  
* brain-calc 
* brain-gcd 
* brain-prime 
* brain-progression
